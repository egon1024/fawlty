Class: Entity¶

This class represents a Sensu entity.

The fields for an entity are:

  • metadata (EntityMetadata)
  • deregister (bool)
  • deregistration (dict)
  • entity_class (str)
  • last_seen (int)
  • redact (list)
  • sensu_agent_version1 (str)
  • subscriptions (list)
  • system1 (dict)
  • user (str)

Example:

from fawlty.resource.clusterrole import ClusterRole

data = {
    "metadata": {
        "name": "test",
        "labels": {"donottalkabout": "thewar"},
        "annotations": {"hotel": "inspectors"},
        "namespace": "torquay",
    }, 
    'entity_class': 'agent',
    'subscriptions': ['DailyExpress'],
    'deregister': False,
    'redact': ['horse_race'],
}

a = Entity(**data)
a.set_client(my_client)
a.create()
